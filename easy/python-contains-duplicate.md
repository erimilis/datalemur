### Contains Duplicates (Python)
* **Topic:** Array & Hash Set Logic
* **Description:** Checks whether there are duplicate values ​​in the list.
* **Approach:** Comparing list length and set length.
* **Links:** [Problem Link](https://datalemur.com/questions/python-contains-duplicate)


### SQL Query
```

def contains_duplicate(input)-> bool:
  ls = list(set(input))

  rslt = False
  if len(input) != len(ls):
    rslt = True

  return rslt


```
